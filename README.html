CLI Snake Game

Terminal-based · Python · Fully configurable

A classic Snake game that runs right in your terminal. Grow your snake by eating fruits, avoid crashing into walls or yourself, and see how high you can score. The game gets faster the more you eat — so don't get too greedy.

Built for Vim learners. The default navigation keys are h j k l — the same keys used to move around in Vim. If you're trying to build that muscle memory, this game is a fun way to drill it in. Every session is practice. By the time you're chasing high scores, those keys will feel second nature.

configurable settings
  • Nav keys — switch to arrow keys or WASD anytime from the config block at the top of the file
  • Speed — tweak the delay between moves to make the game as relaxed or as brutal as you want
  • Fruit count — control how many fruits appear on the board at once
run the game
python3 snake.py
